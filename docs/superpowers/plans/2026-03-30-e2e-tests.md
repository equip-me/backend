# E2E Tests & Test Restructuring Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Restructure the test suite into unit/db/e2e layers and implement comprehensive e2e tests covering all business logic.

**Architecture:** Three-layer test structure: `tests/unit/` (no external deps), `tests/db/` (needs PG, mocks external services), `tests/e2e/` (real everything — PG, MinIO, Redis, Dadata). Root conftest provides shared DB fixtures without mocks. Each layer's conftest adds its own dependency strategy.

**Tech Stack:** pytest, httpx AsyncClient, Tortoise ORM, real Dadata API, real MinIO, real Redis/ARQ, date mocking via `unittest.mock.patch`

**Design spec:** `docs/superpowers/specs/2026-03-30-e2e-tests-design.md`

**Key principle:** If during test implementation the system exhibits incorrect behavior (wrong error code, missing exception), fix the application code — never adjust tests to match broken behavior.

```python
"""E2E tests — real infrastructure: PG, MinIO, Redis, Dadata.

The only mock is date patching for order auto-transitions.
"""

from collections.abc import Generator
from datetime import date
from unittest.mock import patch

import pytest

from app.core.config import get_settings
from app.media.storage import StorageClient


@pytest.fixture
async def real_storage() -> StorageClient:
    """Create a real StorageClient connected to test MinIO."""
    settings = get_settings()
    client = StorageClient(
        endpoint_url=settings.storage.endpoint_url,
        access_key=settings.storage.access_key,
        secret_key=settings.storage.secret_key,
        bucket=settings.storage.bucket,
    )
    await client.ensure_bucket()
    return client


@pytest.fixture
def mock_today() -> Generator[type]:
    """Fixture that returns a context manager for patching date.today().

    Usage:
        with mock_today(date(2026, 5, 1)):
            resp = await client.get(...)
    """

    class _MockToday:
        def __init__(self, target_date: date) -> None:
            self._target = target_date
            self._patcher: Any = None

        def __enter__(self) -> None:
            original_date = date

            class MockDate(date):
                @classmethod
                def today(cls) -> date:
                    return self._target

            self._patcher = patch("app.orders.state_machine.date", MockDate)
            self._patcher.start()

        def __exit__(self, *args: object) -> None:
            if self._patcher:
                self._patcher.stop()

    yield _MockToday
```

Note: The exact module path to patch for `date` depends on where the auto-transition logic imports `date`. Check `app/orders/state_machine.py` during implementation and adjust the patch target accordingly.
